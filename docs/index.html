dotme

One ~/.me folder
every AI tool reads

Set up once. Every AI knows you.

Every AI tool you use — Claude, Cursor, ChatGPT — has amnesia about you. You re-explain who you are, what you're building, and how you like answers, over and over. dotme fixes it with one folder you own: plain markdown, served to every AI through a single local server.
★ GitHub 📦 npm — dotme-ai

60-second quick start

# describe yourself once (~3 min, every question skippable)
$ npx dotme-ai init
# auto-detect your AI tools and hook up every one
$ npx dotme-ai connect all
# done — restart your tools and ask one: "what do you know about me?"

Prefer a permanent install? npm install -g dotme-ai, then use dotme directly.

Why it's safe

Your files

Plain markdown in your home directory. Edit, version, or delete any time. No database, no lock-in.

Your machine

The server runs locally over stdio. Zero network calls, zero telemetry, no accounts. Grep the package — there's no network code.

Private by default

private.md is never exposed to AI unless you opt in. A broken manifest fails closed.

Fully audited

Every AI write lands in .changelog with the tool's name and reason. Make ~/.me a git repo and diff everything.
